# journal/

Time-stamped working notes from porting [rust-by-example](https://github.com/rust-lang/rust-by-example) to Typst documents that evaluate end-to-end through `evcxr-typst`. The port is incremental and exploratory — chapters are written by hand, not by a porter, so the experience of writing each one surfaces things worth remembering.

This directory is the raw record. Polished outputs derived from it live elsewhere:

- **Tutorials** distilled from the journal: `docs/tutorial/`.
- **Tests** that pin behaviours surfaced here: `crates/evcxr-typst/tests/rbe_*.rs`.
- **Decisions** that change the tool itself: `docs/DECISIONS.md`.

See `CLAUDE.md` for the entry shape and naming convention. Entries are sorted chronologically by filename.
